## Описание
Это приложение позволяет пользователям хранить информацию о фильмах, включая название, жанр, год выпуска и рейтинг. Пользователи могут добавлять фильмы в библиотеку и фильтровать их по жанру.

## Примеры использования
1. Введите название фильма, жанр, год выпуска и рейтинг.
2. Нажмите кнопку «Добавить фильм», чтобы сохранить фильм в библиотеке.
3. Используйте поле для фильтрации по жанру и нажмите «Применить фильтр», чтобы отобразить только соответствующие фильмы.
